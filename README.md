# CovDi-Einsatzplanung

Erweiterungs-Modul für CovDi.
Achtung: Es wird ein entsprechendes Formular von FS benötigt, mit welchem die Mitarbeiter ihre Dienste eintragen können. 

Gemeldete Dienste durch Mitarbeiter (die in CovDi einen Account haben) können hier eingeplant werden. 
Folgende Schichten werden in Intrexx benötigt: Fruehschicht, Spaetschicht, Wochenend (die Namen sind bitte so anzuwenden)
In den Schichten können die benötigten Anzahlen an Ärzten, Teamleitungen und Mitarbeitern eingetragen werden, damit ein vollständiger Betrieb gewährleistet wird.

Die freigegebene Schichtplanung kann durch die Mitarbeiter in CovDi eingesehen werden. Die Bearbeitung und Freigabe der EP ist nur mit entsprechenden Rechten möglich.
